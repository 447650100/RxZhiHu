# 知了日报

## 简介

每天3次,每天7分钟,一个纯粹阅读的App.

## 功能介绍

* 日报列表显示
* 日报详情显示
* 主题日报查看
* 热门新闻查看
* 日报评论查看
* 日报主编专栏
* App分享功能
